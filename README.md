# 🚀 Project Management System

A full-stack **Project Management System** developed using **ASP.NET Core Web API**, **Angular**, and **SQL Server**. This application helps organizations efficiently manage users, projects, and tasks through a modern and user-friendly interface.

---

## 📌 Project Overview

The Project Management System is designed to simplify project tracking by providing separate modules for user management, project management, task management, and dashboard analytics. It enables administrators to efficiently organize projects, assign tasks, and monitor progress.

# 🚀 Installation Guide

## Backend

1. Open the backend solution in Visual Studio.
2. Restore NuGet packages.
3. Configure the SQL Server connection string in `appsettings.json`.
4. Run the ASP.NET Core Web API.

---

## Frontend

```bash
npm install
ng serve
```

Open the application in your browser:

```
http://localhost:4200
```

---

## Database Setup

1. Open SQL Server Management Studio.
2. Create a database.
3. Execute the provided SQL script:

```
Database/ProjectManagement.sql
```
